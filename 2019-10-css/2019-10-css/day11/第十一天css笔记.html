什么时候用两套，什么时候用一套？

当页面结构比较简单的时候，我们可以写成一套，当页面结构比较复杂，内容比较多，咱们通常写两套。

pc端和移动端的区别

  • 事件：pc端有鼠标，就有点击、滑过等与鼠标相关的事件，但是移动端没有，它有于手指相关的事件，比如：左滑、右滑等滑屏事件

  • 兼容性：pc端需要处理各个浏览器兼容性问题，但是移动端不需要考虑浏览器兼容问题，因为它的内核和谷歌浏览器一致，但是由于手机屏幕大小不一样，还有手机机型不同，所以我们需要处理适配问题和一些其它兼容性问题。

box-shadow 阴影

  • 第一个值：x轴偏移量

  • 第二个值：y轴偏移量

  • 第三个值：阴影模糊半径

  • 第四个值：阴影大小

  • 第五个值：颜色

  • 第六个值：默认是outset 外阴影 inset 内阴影

.box{

        width:200px;

        height:200px;

        border-radius: 50%;

        background:yellow;

        /* 

         第一个值：阴影x轴偏移量

         第二个值：阴影y轴的偏移量

         第三个值：阴影模糊半径（模糊度）

         第四个值：阴影大小

         第五个值：阴影的颜色

         第六个值：如果写inset 就是内阴影，不写就是外阴影



         */

        box-shadow: 0px 0px 50px 50px darkorange inset;

    }

渐变

  • 线性渐变（Linear Gradients）

  • 径向渐变（Radial Gradients）
    1、线性渐变默认方向是从上向下的（至少两个颜色值）

background:linear-gradient(green,red);

background:-webkit-linear-gradient(green,red);

2、线性渐变的方向（left、right、top、bottom）

从下往上绿色到红色渐变

linear-gradient(bottom,green,red);

3、对角线渐变（方向值可进行组合）

从左上角 到右下角 绿色到红色渐变（对角线）

linear-gradient(left top,green,red);

4、 还可以是具体的角度

linear-gradient(45deg,green,red);

径向渐变

ellipse 椭圆（默认），circle（圆）

radial-gradient(circle, green,red);

视口（viewport）

当页面中不写viewport的时候，在移动端页面默认的宽度是980px;那为了让整个页面在移动设备中全部显示，会把页面进行缩放，这样子里面的内容就看不清楚了。咱们需要加上视口（移动端必须要加的）

<meta name="viewport"

          content="width=device-width, user-scalable=no, initial-scale=1.0, maximum-scale=1.0, minimum-scale=1.0">

  • viewport：视口

  • width=device-width：让页面的宽度等于设备的宽度

  • user-scalable=no：禁止用户缩放

  • initial-scale=1.0：初始缩放值为1.0

  • maximum-scale=1.0:最大的缩放值1.0

  • minimum-scale=1.0:最小的缩放值1.0

媒体查询

media 针对不同媒体类型可以定制不同的样式规则。 例如：你可以针对不同的媒体类型(包括显示器、便携设备、电视机，等等)设置不同的样式规则。 但是这些多媒体类型在很多设备上支持还不够友好。

@media not|only mediatype and (expressions) { CSS 代码...; }

媒体类型

案例1

<!DOCTYPE html>

<html lang="en">

<head>

    <meta charset="UTF-8">

    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <meta http-equiv="X-UA-Compatible" content="ie=edge">

    <title>Document</title>

    <style>

    *{

        margin:0;padding:0;

    }

    .box{

        width:200px;

        height:200px;

        background:green;

    }

    /* 

       当屏幕的宽度小于600px的时候，我就让.box 这个盒子背景颜色pink



     */



     @media screen and (max-width:600px){

         .box{

             background:pink;

         }

     }

     @media screen and (min-width:700px){

         .box{

             background:gold;

         }

     }

     /* 大于等300 小于等于800 */

    @media screen and (min-width:300px) and (max-width:800){



    }



    /* 需求：小于768px的时候显示一个颜色

             大于768到1200 显示一个颜色

             大于1200的时候显示一个颜色 

             */

    </style>

</head>

<body>

    <div class="box"></div>

</body>

</html>

          line-height: 50px;
