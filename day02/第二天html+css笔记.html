第二天html+css笔记

完成一个页面所需要的几门语言

  • html:它是一门超文本标记语言，因为它不仅单纯的可以表示文本，还可以表示音频、视频、图片等。（骨架）

  • css：它是层叠样式表，它主要是用来美化页面的（衣服或者化妆品）

  • js：表示行为和动态的，比如点击一个按钮，鼠标滑动事件等（一个人的行为）

标签的特点

  • 关键词由一对尖括号包裹起来

  • 成对出现

  • 结束比开始标签多了一个”/”

还有一类标签比较特殊，比如下面的img，这类标签叫做 空标签、自闭和标签、单标签

     <p>段落</p>



     <img src="" />

如何快速的创建一个html文档

  1. 文件的后缀名必须是“.html”

  2. 在英文输入法状态下，敲一个！，回车一下

一个html的基本结构

<!DOCTYPE html>

<html>

   <head >

       <meta charset="utf-8">

       <meta name="keywords" content="关键词">

       <meta name="description" content="描述的内容">

       <title>网页标题</title>

   </head>

   <body>

       哈哈哈

   </body>



</html>

标题标签

标题总共可以分为六级，h1 到h6，特点：由粗到细，到大到小，重要性也是如此，h1的重要性最大，剩下的逐次递减。一个页面中通常只出现一次h1，比如logo，外面包括h1的时候更有利于搜索引擎优化。

   //快速生成六级标题的快捷键：h$*6{标题$}

    <h1>标题一</h1>

    <h2>标题二</h2>

    <h3>标题三</h3>

    <h4>标题四</h4>

    <h5>标题五</h5>

    <h6>标题六</h6>

img 标签

  • src:主要存放图片路径

  • alt：当网比较慢或者图片路径不正确，导致图片加载失败，这样的话，用户就不清楚图片是干嘛的，如果你在alt里面写的有描述文字，那这个文字会出现在裂的图片旁边，更有利于用户体验。

  • title：当鼠标滑上图片的时候，会出现title里面设置的文字提示

a标签：超链接

target：

  • _self :在本窗口打开 （默认）

  • _blank ：在新窗口打开

<!-- 

       target: 

        _self 在本窗口打开，默认值

        _blank 在新窗口打开



     -->

   <a href="http://www.baidu.com" target="_self" >百度</a>

a标签中href中的几个作用

  • 跳转的网址：

<a href="http://www.baidu.com">百度</a>

  • 回到顶部：

<a href="#">百度</a>

  • 禁止跳转

<a href="javascript:;">百度</a>

  • 锚点跳转：给每一个模块起一个id名字，然后a标签中的href这个属性值等于#id名字，这样的话，点击a标签能够迅速的跳转到相应的模块

补充的几个标签

  • div：大盒子标签，主要用来划分大区域的

  • span：小盒子标签，主要用来划分小区域的

  • 格式化标签：

    • 加粗：b或者strong

    • 斜体：i或者em

    • 删除线：del

    • 比普通文字增大一个字号：big

    • 比普通文字减小一个字号：small

  • 预格式化标签：pre 浏览器在渲染的时候，会按照预先设置好的格式进行渲染

 <pre>

          ***     ****

            []    []

               ()

 </pre>

三大列表

1、无序列表：ul li

<!-- 无序列表 -->

    <ul>

        <li>上班</li>

        <li>吃饭</li>

        <li>打豆豆</li>

    </ul>

2、有序列表：ol li

 <ol>

        <li>上班</li>

        <li>吃饭</li>

        <li>打豆豆</li>



 </ol>

3、自定义列表

<dl>

        <dt>早餐</dt>

        <dd>豆浆</dd>

        <dd>包子</dd>

        <dt>午餐</dt>

        <dd>盖浇饭</dd>

        <dd>凉菜</dd>

        <dt>晚餐</dt>

        <dd>煎饼</dd>

</dl>

这三大列表是固定格式，ul或者ol里面紧邻着的一定是li，如果想写其它标签，可以写在li里面。 dl里面紧邻着的一定是dt或者dd，不能是其它标签，但是可以在dt或者dd里面写其它的。

标签语义化：

用合适的标签做合适的事情，比如在写一个标题的时候，你也可以用div，但是它仅仅表示一个盒子，并没有标题这个意思，那如果用h1-h6去写的话，浏览一看就知道你这里就是标题，更加具有语义化。
什么时候用列表，一般像商城上面的图片列表，或者是文字列表，咱们都可以用ul或者ol

css 引入的几种方式

1、行内式:要给谁加样式，就直接在谁的开始标签里写style=“…”

 <!-- 第一种方式：行内式 -->

    <div style="width:200px;height:200px;background:gold">box</div>

2、内嵌式（用的比较多），通过style标签，写样式

<!DOCTYPE html>

<html lang="en">

<head>

    <meta charset="UTF-8">

    <title>Document</title>

    <style type="text/css">

     /* 第二种方式：内嵌式 */

      div{

          /* 宽度 */

          width:200px;

          /* 高度 */

          height:200px;

          /* 背景颜色 */

          background:darksalmon;

      }

    </style>

</head>

<body>



    <div>box</div>

</body>

</html>

3、外链式（用的很多，在写大型项目的时候通常都会用这）。通过link标签，把样式表和当前的html页面进行关联

 <!-- 第三种方式:外链式 -->

    <link rel="stylesheet" href="index.css">

4、导入式（用的很少，它是一条命令语句）

 <style>

      /* 第四种方式：导入式 */

      @import "index.css"

</style>

面试题：b或者strong 都表示加粗，i或者em都表示斜体，那它们之间有何区别？

b或者strong ，i或者em 都能达到表象上的加粗或者是斜体，但是 strong 和em带有一定的感情色彩，语气更加强烈，更有利于搜索引擎的优化。
